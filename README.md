## HimoJuku
A ACGN novel reader project

## Developing Status
Currently, the project is still in the developing stage.

## Get started
Install dependencies
```bash
pnpm install
```
Install Expo dependencies
```bash
npx expo install --fix
```
## Start the app
```bash
pnpm exec expo run:android
```
